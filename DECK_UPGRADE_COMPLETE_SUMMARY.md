## Design System Applied

### Typography Scale
```typescript
- H1 (slide titles): text-5xl sm:text-6xl md:text-7xl font-black
- H2 (section headers): text-3xl sm:text-4xl md:text-5xl font-bold
- H3 (sub-headers): text-2xl sm:text-3xl font-bold
- Body: text-base sm:text-lg font-normal
- Labels: text-xs sm:text-sm uppercase tracking-wider
- Metrics: text-4xl sm:text-5xl md:text-6xl font-black
```

### Color System
```typescript
- Primary dark: #0a1628
- Brand gold: #c9a227, gradient: #d4ab2c
- Semantic colors:
  - Success: emerald-500/600
  - Warning: orange-500/600
  - Danger: red-500/600
  - Info: blue-500/600
  - Purple: purple-500/600
  - Teal: teal-500/600
```

### Background Patterns
```typescript
// Animated gradient orbs (all upgraded slides)
<div className="absolute top-0 right-0 w-96 h-96 bg-[color]-500/5 rounded-full blur-3xl animate-pulse" />
<div className="absolute bottom-0 left-0 w-96 h-96 bg-[color]-500/5 rounded-full blur-3xl animate-pulse" style={{ animationDelay: '1s' }} />
```

### Card Styling
```typescript
// Standard card
className="bg-white rounded-3xl p-8 shadow-xl border-2 border-[color]-200 hover:shadow-2xl transition-all duration-300"

// Gradient card
className="bg-gradient-to-br from-[color]-50 to-[color]-100 border-2 border-[color]-300 rounded-3xl p-8"

// Icon badge
className="w-16 h-16 rounded-2xl bg-[color]-500 flex items-center justify-center shadow-lg"
```

### Animation System
```typescript
- Hover scale: hover:scale-110 transition-transform
- Hover translate: hover:-translate-y-2 transition-all duration-300
- Card hover: hover:shadow-2xl transition-all duration-300
- Background orbs: animate-pulse with staggered delays
- Counters: AnimatedNumber component with 1500-2000ms duration
- Stagger delays: style={{ animationDelay: `${idx * 100}ms` }}
```
