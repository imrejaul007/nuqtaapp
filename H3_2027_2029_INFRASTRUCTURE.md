‚ö†Ô∏è **Critical Rule**: You don't become a bank. You become **financial plumbing**.

   - Majid Al Futtaim launches "City Centre Rewards powered by Nuqta"

**Commercial Model**:
- **Setup Fee**: AED 50K-200K per partner
- **Monthly SaaS**: AED 10K-50K depending on scale
- **Transaction Fee**: 0.5-1% of wallet load volume

**Why This Works**:
- **Asset-light** (no balance sheet risk)
- **Recurring revenue** (SaaS model)
- **Data access** (see all spend behavior)

### B. BNPL (Very Limited, Controlled)

‚ö†Ô∏è **Philosophy**: BNPL is a **feature**, not identity.

**Who Gets BNPL**:

1. **Verified Employees**:
   - Users with direct salary deposits
   - 3-month salary history
   - Employer verification

2. **Students with Stipends**:
   - University-verified enrollment
   - Scholarship/stipend confirmation
   - Parent guarantor for international students

**What Categories**:
- ‚úÖ Education (course fees, books)
- ‚úÖ Healthcare (dentist, physiotherapy packages)
- ‚úÖ Fitness (gym memberships, sports equipment)
- ‚ùå F&B (too low LTV)
- ‚ùå Fashion (high return rate)
- ‚ùå Electronics (fraud risk)

**How It Works**:
```
User wants AED 1,200 dental package at NMC Dental
‚Üí Nuqta checks: Salary history, spending pattern, fraud score
‚Üí Offers: 3-month BNPL (AED 400/month, 0% interest)
‚Üí User accepts
‚Üí NMC gets full AED 1,200 upfront (Nuqta takes risk)
‚Üí Nuqta collects: AED 400/month for 3 months from salary account
‚Üí If default: Collection via employer deduction (pre-authorized)
```

**Commercial Model**:
- **Merchant Fee**: 5-7% (vs 3-5% for regular transactions)
- **No user interest** (merchant pays the fee)
- **Default rate target**: <2% (very selective underwriting)

**Why This Is Safe**:
- **Salary-linked repayment** (auto-deduct from salary account)
- **Contextual only** (education, health, fitness - high completion categories)
- **Employer partnerships** (pre-authorized deduction agreements)

üìå **BNPL contributes <10% of revenue**. This is NOT a BNPL company.
