## Design System Applied

### **Color Palette**
- **White/Light**: Default backgrounds
- **Gradients**: Subtle color washes (red-50, orange-50, blue-50, emerald-50, purple-50, gold-10)
- **Accent Colors**:
  - Gold (#c9a227) - Brand
  - Red - Problems/urgency
  - Orange - Warnings
  - Emerald - Success/positive
  - Blue - Trust/information
  - Purple - Vision/future
  - Slate - Neutral

### **Typography Scale**
- **Display**: text-8xl, text-9xl (Cover, closing)
- **Hero Numbers**: text-6xl, text-7xl (Metrics)
- **Headings**: text-5xl, text-6xl (Slide titles)
- **Subheadings**: text-2xl, text-3xl (Section headers)
- **Body**: text-base, text-lg, text-xl
- **Labels**: text-xs (uppercase, tracking-wider)

### **Layout Patterns**
1. **Hero Cover** - Centered, large typography
2. **Dashboard Split** - Metrics + content grid
3. **Horizontal Cards** - Icon left, content right
4. **Comparison** - Side-by-side columns
5. **Timeline** - Vertical stacked cards
6. **Grid Systems** - 2x2, 3-column, 4-column

### **Component Patterns**
- **Icon Boxes**: Colored background, rounded
- **Metric Cards**: Border accents, animated numbers
- **Feature Cards**: Icon + title + description
- **Gradient Cards**: Subtle backgrounds, shadows
- **Badges**: Rounded-full pills
- **Stat Bars**: Dark backgrounds for contrast
