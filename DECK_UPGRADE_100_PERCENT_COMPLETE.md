```typescript
// Standard card
className="bg-white rounded-3xl p-8 shadow-xl border-2 border-[color]-200 hover:shadow-2xl transition-all duration-300"

// Gradient card
className="bg-gradient-to-br from-[color]-50 to-[color]-100 border-2 border-[color]-300 rounded-3xl p-8"

// Icon badge
className="w-16 h-16 rounded-2xl bg-[color]-500 flex items-center justify-center shadow-lg"
```

### Animation System
```typescript
- Hover scale: hover:scale-110 transition-transform
- Hover translate: hover:-translate-y-2 transition-all duration-300
- Card hover: hover:shadow-2xl transition-all duration-300
- Background orbs: animate-pulse with staggered delays
- Counters: AnimatedNumber component with 1500-2000ms duration
- Stagger delays: style={{ animationDelay: `${idx * 100}ms` }}
```
